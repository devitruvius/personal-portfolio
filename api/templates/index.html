Systems Analysis and Development

Sobre

Sou graduando em Análise e Desenvolvimento de Sistemas pela UFCA e estou buscando complementar minha formação profissional com experiência prática em desenvolvimento de Back-end. Minha expertise está na criação de APIs robustas e eficientes utilizando Python e Flask. Além disso, possuo um forte interesse em pesquisa científica e desejo contribuir como pesquisador para estudos que impulsionem avanços na área de software.

Projeto de Extensão

Como voluntário no Grupo de Estudos em Sistemas de Informação e Inovação Digital da UFCA, estou colaborando no desenvolvimento de uma API pública. Nosso objetivo é democratizar o acesso à informação e ao conhecimento na área de software, beneficiando pesquisadores, estudantes e profissionais. Dentre as atividades desenvolvidas, destaco:

  • Comunicação e Alinhamento: Definição clara de objetivos e expectativas.
  • Configuração do Ambiente: Instalação de ferramentas essenciais para o projeto.
  • Swagger: Documentação detalhada da API, garantindo clareza e acessibilidade.
  • Flask: Construção de uma API robusta e escalável, otimizada para performance.
  • Metodologias ágeis (SCRUM): Foco em entregas iterativas.
  • Análise de Dados: Coleta e formatação de informações relevantes em planilhas.
  • Implementação de Soluções: Criação de rotas eficientes e eficazes para a API.
  • Testes Unitários: Garantia da qualidade da API através de testes rigorosos.

Habilidades

Domínio em Python para desenvolvimento de APIs RESTful, garantindo acesso e manipulação de dados de forma rápida e eficiente e testes unitários para garantir a confiabilidade do código. Também possuo habilidades em tratamento de dados e criação de scripts para automação. Para mais detalhes, consulte meu GitHub: https://github.com/devitruvius.

Portfólio

College Repository

Este repositório reúne minhas atividades e aprendizados na graduação em Análise e Desenvolvimento de Sistemas (ADS) pela UFCA. Aqui você encontrará códigos, relatórios, documentações e outros materiais relevantes, organizados por semestre e disciplina.

Snake Game

Controle a cobra, coma comida e evite colisões! O jogo utiliza Pygame para gráficos, eventos e lógica. Este projeto demonstra minhas habilidades em programação de jogos, manipulação de eventos e implementação de lógica de jogo.

Portfolio Website

Utilizando o Flask em Python e Bootstrap para estilização, a aplicação permite que eu apresente meus projetos, habilidades e outras informações de forma interativa. Também inclui um formulário de contato que envia emails automaticamente para meu endereço de email.

Budget Generator

Ferramenta automatizada para gerar orçamentos de projetos em PDF. A partir da descrição do projeto, horas estimadas, valor da hora trabalhada e prazo estimado, o script calcula o valor total estimado e gera um PDF com as informações detalhadas.

Yahoo Finances Automation

Script Python que automatiza a análise de dados de ações e envio de relatórios por email. Usa as bibliotecas yfinance para obter dados de preços, pyautogui para automatizar interações com o Gmail e pyperclip para colar dados entre a aplicação e a área de transferência.

Tax Calculation System

Programa em C# para calcular o imposto de renda. O sistema considera o tipo de contribuinte (física ou jurídica), suas informações pessoais (nome, endereço, CPF ou CNPJ e IE) e o valor da compra. Em seguida, calcula o imposto devido e exibe o valor total a ser pago.

Amazon Product Scraper

Uma aplicação web simples para extrair listagens de produtos da Amazon da primeira página de resultados de busca para uma palavra-chave específica. O backend é construído com Node.js e Express, e o frontend utiliza HTML, CSS e JavaScript puro (Vanilla JavaScript).

MTG Codex

Aplicação web para buscas avançadas em cartas de Magic. Utiliza Flask e a API do Scryfall para o Back-end e HTML, CSS e JavaScript para o Front-End. Os resultados são exibidos como cartas com detalhes como nome, tipo, texto oracular e imagem representativa.
