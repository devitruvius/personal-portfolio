Sobre

Sou graduando em ADS pela UFCA e meu objetivo é me tornar um Arquiteto de Software de excelência. Acredito no poder transformador da tecnologia e estou constantemente aprimorando minhas habilidades em Linguagens de Programação e Desenvolvimento Web, Análise e Especificação de Requisitos (IEEE Std 830), Banco de Dados e Inteligência Artificial. Entusiasta por Desenvolvimento de Jogos, estou me dedicando à criação do meu primeiro jogo com a Godot Engine, com o objetivo de criar uma experiência imersiva e transformadora.

Voluntariado

Atualmente sou voluntário no Grupo de Estudos em Sistemas de Informação e Inovação Digital (GESID) da UFCA, contribuindo para um Projeto de Extensão envolvendo o desenvolvimento de uma API pública utilizando o Flask. Essa API deve facilitar o acesso à informação para pesquisadores, estudantes e profissionais da área de software.

  • Comunicação e Alinhamento: Definir objetivos e expectativas.
  • Configurar o Ambiente: Instalação do Python e demais ferramentas.
  • Swagger: Para documentar a API de forma eficaz.
  • Flask: Para construção de uma API robusta e escalável.
  • Gerenciamento de Projeto: Metodologias ágeis (SCRUM).
  • Curadoria de Conteúdo: Coletar e formatar informações numa planilha.
  • Implementação de Soluções: Criar as rotas e a lógica da API.
  • Testes Unitários: Garantir a qualidade e confiabilidade da API.

Habilidades

Expertise em Python com foco em Flask, já tive contato com algumas linguagens, como Angular, TypeScript, .NET e C#. Essa versatilidade permitiu que eu me adaptasse a diferentes tecnologias e a entregar soluções de qualidade, alinhadas às necessidades específicas de cada projeto.
