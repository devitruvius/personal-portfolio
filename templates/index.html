Voluntariado

Essa API deve facilitar o acesso à informação para pesquisadores, estudantes e profissionais da área de software.

  • Comunicação e Alinhamento: Definir objetivos e expectativas.

  • Swagger: Para documentar a API de forma eficaz.

  • Gerenciamento de Projeto: Metodologias ágeis (SCRUM).
  • Curadoria de Conteúdo: Coletar e formatar informações numa planilha.
  • Implementação de Soluções: Criar as rotas e a lógica da API.
