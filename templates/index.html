Systems Analysis and Development

Sobre

Graduando em Análise e Desenvolvimento de Sistemas pela UFCA com o objetivo de me tornar especialista em Data Analytics e impulsionar o crescimento e a inovação com soluções de alto impacto. Sou capacitado para construir soluções com Python e ferramentas de IA e atualmente estou me dedicando ao estudo de SQL para manipulação e extração de informações de diversas fontes de dados (bancos relacionais, NoSQL e Big Data).

Voluntariado

Sou voluntário no GESID pela UFCA e contribuo para o desenvolvimento de uma API pública. O objetivo é facilitar o acesso à informação e democratizar o conhecimento na área de software, beneficiando pesquisadores, estudantes e profissionais. Através do projeto, estou aprimorando minhas habilidades em Python, além de outras competências, como:

  • Comunicação e Alinhamento: Para definição de objetivos e expectativas.
  • Configuração do Ambiente: Para instalação das ferramentas.
  • Swagger: Para documentação em detalhes da API.
  • Flask: Para construção de uma API robusta e escalável.
  • Metodologias ágeis (SCRUM): Para gerenciamento de projeto.
  • Curadoria de Conteúdo: Para coleta e formatação de informações na planilha.
  • Implementação de Soluções: Na criação de rotas da API.
  • Testes Unitários: Para garantia da qualidade e confiabilidade da API.

Habilidades

Python para o desenvolvimento de API Restful com integração a bancos de dados, possibilitando acesso e manipulação de dados com agilidade e eficiência. Domínio básico de Angular e TypeScript para criação de interfaces web interativas que facilitam a visualização e análise de dados. Conhecimento básico em C# para desenvolvimento de sistemas. Para mais detalhes, consulte meu GitHub: https://github.com/devitruvius.

Portfólio

College Repository

Este repositório reúne minhas atividades e aprendizados na graduação em Análise e Desenvolvimento de Sistemas (ADS) pela UFCA. Aqui você encontrará códigos, relatórios, documentações e outros materiais relevantes, organizados por semestre e disciplina.

Snake Game

Controle a cobra, coma comida e evite colisões! O jogo utiliza Pygame para gráficos, eventos e lógica. Este projeto demonstra minhas habilidades em programação de jogos, manipulação de eventos e implementação de lógica de jogo.

Portfolio Website

Utilizando o Flask em Python e Bootstrap para estilização, a aplicação permite que eu apresente meus projetos, habilidades e outras informações de forma interativa. Também inclui um formulário de contato que envia emails automaticamente para meu endereço de email.

Budget Generator

Ferramenta automatizada para gerar orçamentos de projetos em PDF. A partir da descrição do projeto, horas estimadas, valor da hora trabalhada e prazo estimado, o script calcula o valor total estimado e gera um PDF com as informações detalhadas.

Yahoo Finances Automation

Script Python que automatiza a análise de dados de ações e envio de relatórios por email. Usa as bibliotecas yfinance para obter dados de preços, pyautogui para automatizar interações com o Gmail e pyperclip para colar dados entre a aplicação e a área de transferência.

Tax Calculation System

Programa em C# para calcular o imposto de renda. O sistema considera o tipo de contribuinte (física ou jurídica), suas informações pessoais (nome, endereço, CPF ou CNPJ e IE) e o valor da compra. Em seguida, calcula o imposto devido e exibe o valor total a ser pago.

Landing Page Template

O projeto demonstra minhas habilidades na criação de aplicações web modernas e responsivas utilizando o framework Angular. A aplicação foi desenvolvida seguindo as melhores práticas e padrões do Angular CLI, garantindo um código limpo, modular e bem documentado.

Public API

Contribuindo para o desenvolvimento de uma API RESTful em Flask para gerenciar um banco de dados de artigos científicos. A API permite a pesquisa de artigos por diversos critérios e a recuperação de resumos, referências e citações.
